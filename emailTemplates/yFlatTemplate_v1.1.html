Instructions

Ce template défini des styles pour:

  • Le corps de texte
  • Les titres de niveau 1 à 3
  • Les règles horizontales
  • Les listes à puces
  • La signature

Pour modifier le code HTML: Ctrl+A --> Insérer HTML. Depuis là, tout est modifiable.

Pour ajouter une ligne horizontale: Insérer --> Ligne Horizontale ou Ctrl+A --> Insérer --> HTML --> "<hr>" à l'endroit désiré dans le texte.

La signature doit se trouver dans un bloc <div class=""signature> Signature </div>.

Note: L'ensemble du texte doit être contenu dans un bloc <div id="page-wrap"> afin que la largeur du texte reste agréable pour la lecture.
